# AWS Docs — VELA

- `deploy_quickstart.md` — minimal steps to deploy the backend
- `api_mapping.md` — detailed API mapping (added in Step 2)
- `functions/` — per-lambda references (added in Step 3)
- `dynamodb.md` and `auth_and_cognito.md` — data model and auth docs (Step 4)
- `diagrams/` — mermaid diagrams and call graphs (Step 5)

- `known_gaps.md` — discovered missing artifacts and follow-ups (Step 6)

Follow the Step-by-Step sections in `implementation.md` to iteratively add and commit these files.
